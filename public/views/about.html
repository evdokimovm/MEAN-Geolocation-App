AngularJS Geolocation App

This AngularJS app shows your current location via the lat and lon coordinates and displays on the map and also shows some of the sights from Wikipedia API and from built-in Node.js JSON API

You can add geolocation point to map by select point on map and then move to Add Geo page to fill form (info about added point latitude and longitude added to form automatically after select point on map) and add this point to map
